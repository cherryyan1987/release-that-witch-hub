---

## 🎬 About the project

Release That Witch started as a popular web novel and later expanded into manhua and anime adaptations.

Because each version presents the story differently, this hub was created to make it easier to:

- Compare manga and anime
- Understand character development
- Follow the story structure more clearly

---

## 🔍 Why this exists

There wasn’t a clean place that connects:

- manga reading
- anime adaptation
- character guides
- story explanations

So this project is a simple attempt to bring everything into one place.
